## 🛠️ How to Use
1. **Clone the repository** 🖥️
   ```bash
   git clone https://github.com/avirup010/Coding-Questions.git
   ```
2. **Navigate to the desired folder** 📂
   ```bash
   cd Daily-Coding-Solutions/LeetCode/easy
   ```
3. **Run the solution in your preferred language** ⚡
